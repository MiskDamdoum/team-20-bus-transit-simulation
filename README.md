# Team 20 – Bus Transit Simulation

## Software Fundamentals Final Project

## Project Description

An object-oriented bus route simulator designed to evaluate scheduling strategies on an 18-stop loop route. The system models passenger arrivals, boarding and alighting times, bus capacity limits, stop dwell behavior, schedule delays, stop-skipping rules, and bus queueing at occupied stops.

The project consists of three programs:
- **Passenger Arrival Generator**
- **Route Generator**
- **Simulation Engine**

Together, these programs allow the same passenger demand pattern to be tested across different configurations, including:
- Number of buses
- Departure headways
- Travel times between stops
- Bus capacities

The simulation measures key performance metrics such as:
- Average passenger wait time
- Average trip time (origin to destination)
- Bus load factor at each stop
- Number of passengers left behind due to full buses
- Schedule adherence for each bus
