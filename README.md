# team-20-bus-transit-simulation
Software Fundamentals Final Project

Project Description: 
-An object-oriented bus route simulator designed to evaluate scheduling strategies on an 18-stop loop route. The system models passenger arrivals, boarding/alighting times, bus capacity limits, stop dwell behavior, schedule delays, stop-skipping rules, and bus queueing at occupied stops.
-The project consists of three programs: a Passenger Arrival Generator, a Route Generator, and a Simulation Engine. Together, they allow the same passenger demand pattern to be tested across different configurations (number of buses, departure headways, travel times, and capacities).
-The simulation measures key performance metrics including average passenger wait time, average trip time, bus load factor, passengers left behind due to full buses, and schedule adherence.
